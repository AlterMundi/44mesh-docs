## Networking Protocols

- **BGP RFC 4271**: [rfc-editor.org/rfc/rfc4271](https://www.rfc-editor.org/rfc/rfc4271) — A Border Gateway Protocol 4
- **BIRD2 Documentation**: [bird.network.cz](https://bird.network.cz/) — Internet routing daemon used for BGP
- **ZeroTier Documentation**: [docs.zerotier.com](https://docs.zerotier.com/) — Official ZeroTier documentation

## IP Address Resources

- **AMPRNet (44/8)**: [ampr.org](https://www.ampr.org/) — Amateur Radio IP network; free `/24` allocations for licensed operators
- **ARIN**: [arin.net](https://www.arin.net/) — RIR for North America
- **LACNIC**: [lacnic.net](https://www.lacnic.net/) — RIR for Latin America and Caribbean
- **RIPE NCC**: [ripe.net](https://www.ripe.net/) — RIR for Europe, Middle East, and Central Asia
- **APNIC**: [apnic.net](https://www.apnic.com/) — RIR for Asia-Pacific
- **AFRINIC**: [afrinic.net](https://www.afrinic.net/) — RIR for Africa

## BGP Looking Glasses and Tools

- **BGP Tools**: [bgp.tools](https://bgp.tools/) — ASN and prefix lookup, routing table visibility
- **Hurricane Electric BGP Toolkit**: [bgp.he.net](https://bgp.he.net/) — BGP prefix and AS information
- **RIPE RIS**: [ris.ripe.net](https://ris.ripe.net/) — Routing information service; check if your prefix is visible globally
